Línea de Tiempo de Tecnologías Web

Hecho por Jasser Ramos

Haga un clic en las imagenes si desea mas informacion.

JUL 24, 1989
CREACION DEL LENGUAJE HTML

En 1989, Tim Berners-Lee y Anders Berglund, dos investigadores del Laboratorio Europea de Física de Partículas, crearón un lenguaje basado en etiquetas e hipertexto para marcar documentos técnicos con el fin de compartirlos en Internet.

NOV 16, 1991
NACE EL PRIMER NAVEGADOR WEB "NEXUS 0.1"

Nace el primer navegador web, llamado WorldWideWeb y más tarde rebautizado Nexus, era un navegador de Web editor. Podía ser utilizado para crear páginas, así como navegar por ellas.

APR 22, 1993
ES CREADO EL NAVEGADOR "MOSAIC"

Fue el primer navegador web gráfico disponible para visualizar páginas web en el sistema operativo Microsoft Windows.

JUN 8, 1994
LANZAMIENTO DE HTML2.

Editado por Tim Berners Lee y Dan Conolly, es la primera versión oficial del lenguaje informático HTML, y supone una formalización de las anteriores versiones de HTML

MAR 9, 1995
ES LANZADO HTML 3.0

Tras diferentes conversaciones entre Tim Berners-Lee y el MIT, el 1 de octubre de 1994 es creado el consorcio W3. Más conocido como W3C (World Wide Web Consortium) con la idea de definir estándares para Internet. Dentro de la W3C se empieza a trabajar en noviembre 1995 sobre el borrador de HTML 3.0, el cual nunca llegará a ser recomendación y se quedará en borrador. Es HTML 3.2 la versión que pasa a ser la recomendación.

NOV 12, 1996
LANZAMIENTO DE FLASH 1.0

Un multimedia de la plataforma de software utilizado para la reproducción de animaciones, juegos de navegador, aplicaciones de Internet, aplicaciones de escritorio, aplicaciones móviles y juegos para móviles .

DEC 17, 1996
LANZAMIENTO DE LA PRIMER TECNOLOGIA CSS "CSS 1"

Hojas de estilo en cascada (Cascading Stylesheets) un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

FEB 12, 1997
LANZAMIENTO DE FLASH EN SU SEGUNDA VERSION

DEC 26, 1997
LANZAMIENTO DE ASP

Microsoft introdujo la tecnología llamada Active Server Pages en diciembre de 1997.

FEB 21, 1998
LANZAMIENTO DE HTML4

El lenguaje HTML 4.0 es una especificación del lenguaje html aprobada por el World Wide Web Consortium.

MAY 30, 1998
LANZAMIENTO DE CSS2

La especificación CSS2 fue desarrollada por la W3C y publicada como recomendación en mayo de 1998, y abandonada en abril de 2008.

JUN 16, 1998
LANZAMIENTO DE PHP3

Se creo la base del PHP3, y cambiaron el nombre del lenguaje por PHP: Hypertext Preprocessor. y se publicó oficialmente en junio de 1998.

AUG 12, 1998
LANZAMIENTO DE FLASH3

JUL 13, 2004
LANZAMIENTO DE PHP 5

El 13 de julio del 2004, se lanzó PHP 5, utilizando el motor Zend Engine 2.0

JUL 30, 2005
LANZAMIENTO DE DJANGO

Un framework de desarrollo web de código abierto, escrito en Python, que respeta el patrón de diseño conocido como Modelo–vista–controlador.

AUG 6, 2005
LANZAMIENTO DE AJAX

Un framework para aplicaciones web que ayuda a desarrollar aplicaciones basadas en AJAX, una tecnología para construir páginas web dinámicas del lado del cliente.

SEP 7, 2007
LANZAMIENTO DE MICROSOFT SILVERLIGHT

Una estructura para aplicaciones web que agrega nuevas funciones multimedia como la reproducción de vídeos, gráficos vectoriales, animaciones e interactividad, en forma similar a lo que hace Adobe Flash.

JUN 23, 2010
LANZAMIENTO DE FLASH 10

Versión 10.1.53.64: junio de 2010.

DEC 12, 2011
LANZAMIENTO DE CSS3

CSS3 es la última evolución del lenguaje de las Hojas de Estilo en Cascada (Cascading Style Sheets).

OCT 12, 2014
LANZAMIENTO DE HTML5

Es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 especifica dos variantes de sintaxis para HTML: una «clásica», HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML5 que deberá servirse con sintaxis XML (application/xhtml+xml).
